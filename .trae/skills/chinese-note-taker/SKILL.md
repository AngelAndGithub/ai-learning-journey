# 中文笔记助手

这个技能帮助用户创建和管理中文学习笔记，支持中文内容的组织和保存，并且可以生成ipynb格式的笔记。

## 使用场景

当以下情况时调用此技能：
- 用户要求生成中文笔记
- 用户需要保存中文学习内容
- 用户希望组织中文学习材料
- 用户请求笔记记录帮助
- 用户需要ipynb格式的学习笔记

## 功能特性

- **创建结构化中文笔记**：使用清晰的标题和章节组织内容
- **生成ipynb格式笔记**：支持创建Jupyter Notebook格式的笔记
- **管理中文学习材料**：按主题或学科组织笔记
- **保存中文笔记**：为将来参考保存学习内容
- **格式化中文笔记**：以干净、易读的风格呈现笔记
- **按学习进度命名**：根据学习进度和知识要点自动命名笔记

## 使用示例

### 示例1：生成学习笔记

```
用户：你能帮我创建关于数据分析的笔记吗？
助手：我将帮你创建关于数据分析的综合笔记。让我组织关键概念...
```

### 示例2：生成ipynb格式笔记

```
用户：请生成ipynb格式的机器学习笔记。
助手：我将帮你生成ipynb格式的机器学习笔记，包含代码示例和解释...
```

### 示例3：按学习进度命名笔记

```
用户：将数据分布分析详解记录为笔记，按照学习进度命名。
助手：我将创建关于数据分布分析的笔记，并按照学习进度和知识要点命名...
```

## 笔记结构

创建笔记时，遵循以下结构：

1. **标题**：清晰描述性标题
2. **简介**：主题的简要概述
3. **主要部分**：按关键概念组织
4. **要点**：突出重要信息
5. **示例**：包含相关示例和代码
6. **总结**：简洁回顾
7. **额外资源**：可选参考资料

## 目录和命名规则

### 目录结构

笔记将存储在用户现有的目录结构中：

```
notebooks/
├── phase01/
├── phase02/
├── phase03/
└── phase04/
```

### 命名规则

笔记文件命名按照学习进度和知识要点进行命名：

```
dayXX-主题-知识要点.ipynb
```

其中：
- `dayXX`：学习天数，如day25
- `主题`：笔记的主题，如pandas-advanced
- `知识要点`：笔记的核心内容，如data-distribution

### 示例命名

```
day25-pandas-advanced-data-distribution.ipynb
day29-data-analysis-process-data-visualization.ipynb
```

## IPYNB格式特性

- **代码单元格**：包含可执行的代码示例
- ** markdown单元格**：包含解释和说明
- **富文本支持**：支持图片、表格等富文本内容
- **交互性**：可以在Notebook中直接运行代码

## 最佳实践

- 保持笔记简洁明了
- 使用清晰的标题和子标题
- 包含相关示例和解释
- 随着学习进展定期更新笔记
- 将笔记存储在一致的位置以便于访问
- 为每个笔记添加适当的标签和目录
