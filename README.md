# 90天AI开发工程师系统化全链路学习项目 🚀

## 项目目标

从零起步，循序渐进完成从 Python 基础 → 数据分析 → 机器学习 → 深度学习 → 大模型LLM开发 & 微调 & 部署落地 全栈能力闭环，
系统化掌握 AI 开发工程师核心必备技能，具备独立开发AI应用、微调大模型、搭建智能服务、落地商业化AI项目的能力。

## 适用人群

零基础/转行/计算机基础薄弱，想要系统入门AI开发、大模型应用开发的学习者

## 整体周期

总周期：90天，分5大阶段，阶梯式递进，理论+实操+作业+实战项目结合
1. 第一阶段：Python 编程基础（1~20天）
2. 第二阶段：数据分析 & 数学基础（21~35天）
3. 第三阶段：传统机器学习核心（36~55天）
4. 第四阶段：深度学习 & 神经网络（56~75天）
5. 第五阶段：大语言模型LLM应用开发+微调+部署（76~90天）

## 核心技术栈规划

### 1. 编程基础
Python 基础语法、面向对象、文件操作、异常处理、模块化编程、Git 基础、Linux 常用命令

### 2. AI 基础工具
Numpy、Pandas、Matplotlib、Seaborn、Jupyter Notebook、Conda 环境管理

### 3. 机器学习
sklearn、特征工程、分类/回归/聚类、模型评估、交叉验证、特征筛选、经典算法原理与实战

### 4. 深度学习
Pytorch 框架、张量运算、卷积神经网络CNN、循环神经网络RNN、激活函数、损失函数、优化器

### 5. 大模型AI开发
LangChain、LlamaIndex、Prompt工程、RAG检索增强生成、Agent智能体、
大模型轻量化微调（LoRA/QLoRA）、本地大模型部署、API 服务封装、Web AI 应用搭建

## 各阶段核心学习内容&产出要求

### 阶段一：Python 基础（1-20天）
1. **Day 1-4**: 基础语法（变量、数据类型、循环、条件、函数）
2. **Day 5**: 面向对象编程（类、继承、多态、特殊方法）
3. **Day 6**: 文件操作（文本文件、二进制文件、目录操作）
4. **Day 7**: 异常处理（try-except、自定义异常、最佳实践）
5. **Day 8**: 装饰器（函数装饰器、参数化装饰器、实用案例）
6. **Day 9**: JSON/XML解析（数据序列化/反序列化）
7. **Day 10**: Git基础（版本控制、提交、分支）
8. **Day 11**: Linux命令（文件管理、系统信息、网络操作）
9. **Day 12-20**: 项目实践（计算器、批量重命名、日志工具、网络爬虫、文件处理器、数据分析工具、密码生成器、文本处理器）
10. **阶段产出**：10个实用小工具，形成完整的Python基础技能体系

### 阶段二：数据分析+AI数学基础（21-35天）
1. **Day 21**: NumPy基础（数组创建、操作、基本函数）
2. **Day 22**: NumPy高级（向量化、线性代数、性能优化）
3. **Day 23**: Pandas基础（DataFrame创建、操作、基本功能）
4. **Day 24**: Pandas数据清洗（缺失值处理、异常值检测）
5. **Day 25**: Pandas高级（分组、聚合、时间序列分析）
6. **Day 26**: Matplotlib基础（折线图、柱状图、散点图）
7. **Day 27**: Matplotlib高级（3D绘图、动画、自定义样式）
8. **Day 28**: Seaborn基础（统计数据可视化）
9. **Day 29**: 数据分析流程（数据获取、清洗、分析、报告）
10. **Day 30**: 缺失值处理（删除、插值、模型预测）
11. **Day 31**: 异常值检测（统计方法、机器学习方法）
12. **Day 32**: 特征工程（特征创建、转换、选择、降维）
13. **Day 33**: 数据可视化（Matplotlib、Seaborn、Plotly）
14. **Day 34**: 数学基础-线性代数（向量、矩阵、线性方程组、SVD、PCA）
15. **Day 35**: 数学基础-概率统计（概率、随机变量、统计量、假设检验）
16. **阶段产出**：2份完整行业数据分析报告，掌握数据分析全流程

### 阶段三：机器学习实战（36-55天）
1. **Day 36**: 机器学习基础概念（概述、数据准备、监督/无监督学习）
2. **Day 37**: 监督学习-分类（逻辑回归、K近邻、SVM、决策树、随机森林、梯度提升）
3. **Day 38**: 监督学习-回归（线性回归、岭回归、Lasso回归、随机森林回归、梯度提升回归）
4. **Day 39**: 无监督学习（K-means、DBSCAN、层次聚类、PCA、t-SNE、异常检测）
5. **Day 40**: 模型评估与调优（交叉验证、网格搜索、特征重要性、过拟合处理）
6. **Day 46**: 实战项目1 - 房价预测（数据探索、特征工程、模型训练与评估）
7. **Day 47**: 实战项目2 - 用户分群（聚类算法、结果分析、客户画像）
8. **Day 48**: 实战项目3 - 情感分类（文本预处理、特征提取、分类模型）
9. **阶段产出**：3个完整机器学习实战项目，掌握模型训练与部署全流程

### 阶段四：深度学习入门（56-75天）
1. Pytorch 全套使用、数据集加载、自定义网络搭建
2. CNN图像分类、图像预处理、数据增强
3. 神经网络训练原理、反向传播、学习率调整
4. 阶段产出：图像识别分类项目、自定义神经网络训练案例

### 阶段五：大模型LLM开发落地（76-90天）
1. **Day 76-80**: 大模型基础（概述、核心技术、应用场景、评估指标）
2. **Day 81-85**: 大模型应用开发（Prompt工程、RAG检索增强、向量数据库）
3. **Day 86-90**: 大模型部署与微调（本地部署、LoRA轻量化微调、API服务封装）
4. **阶段终极产出**：个人专属AI知识库+RAG问答系统（可本地运行）

## 常见问题

### Q1: 如何安装Anaconda？
A: 访问官网下载对应系统的安装包，按提示安装即可。

### Q2: Jupyter Notebook无法启动？
A: 尝试命令 `jupyter notebook --no-browser`，然后手动打开浏览器访问显示的URL。

### Q3: 代码运行报错怎么办？
A: 
1. 仔细阅读错误信息
2. 检查语法是否正确
3. 搜索错误信息找解决方案
4. 记录到学习笔记中
